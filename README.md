# Austine Dev Portfolio

A responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This project showcases the portfolio of Austine Dev, an IT student and aspiring software developer. It includes:

- A hero section with animated typing effect
- About section with personal summary and education details
- Skills section with current tech stack
- Projects section with sample portfolio work
- Live sorting visualizer demo section
- Contact section with email, GitHub, and LinkedIn links
- Smooth scrolling, reveal animations, and interactive card tilt effects

## Project Structure

- `index.html` — main portfolio page
- `style.css` — styling and layout
- `script.js` — interactive JavaScript features
- `images/` — image assets used on the site
- `weather-app/` — standalone WeatherNow app with real weather search

## Additional Project

This portfolio links to a separate Weather app repo:
- [WeatherNow App](https://github.com/Austin-dev7/Weather-App)

## Live Preview

To preview locally, open `index.html` in a browser or use VS Code Live Server.

## WeatherNow App Local Run

1. Open `weather-app/` in VS Code.
2. Open `weather-app/index.html` in your browser or use Live Server.
3. Search any city and view live weather data.

## How to Use

1. Open the project folder in VS Code.
2. Start Live Server or open `index.html` directly.
3. Edit content in `index.html`, `style.css`, or `script.js` as needed.
